### 1. &nbsp; Background of Development <br/> 
- The proportion of adult smartphone use in Korea reached 95%, and accordingly, I thought it was necessary to change the drinking game culture. (Analog Drinking Game => Digital Drinking Game). So I developed Alcohol Mini Game App that makes drinking parties fun.<br/> <br/> <br/> <br/>

### 3. &nbsp; Main Function <br/> <br/> 
- Crocodile Game <br/> <br/> 
<img src="https://github.com/qortmdgh4141/Alcohol_Mini_Game_App/blob/main/image/crocodile_game.png?raw=true"  width="720" height="240"> <br/>
- Click on the teeth of a sleeping crocodile. 
- If you click on the tooth that wakes a sleeping crocodile, it will appear!
- In that case, you should drink a boilermaker. <br/> <br/> <br/> <br/> 

- Up-Down Game <br/> <br/>
<img src="https://github.com/qortmdgh4141/Alcohol_Mini_Game_App/blob/main/image/up_down_game.png?raw=true"  width="800" height="480"> <br/>
- Up-down Game is a game in which players guess random numbers from 1 to 100.
- If the number set by the application is less than the number said by the player, the application says 'up', and if it is large, the application gives a hint by saying 'down'.
- All players except the player who got the correct answer must drink boilermaker. <br/> <br/> <br/> <br/>
